Problems

1. Fibonacci

Enter a number to run the Fibonacci sequence that many times.



2. Bubble Sort

Enter a list of numbers to sort, no spaces.



3. Reverse String

Enter a string to reverse.



4. Factorial

Enter the size of the factorial to compute.



5. Substring

Enter a string to get a substring from.

Input the length of the desired substring.

Input the left offset for the substring to start.





6. Even Number

Enter a number to check if it's even.



7. Palindrome

Enter a string to check if it's a palindrome.



8. Shapes

Choose one of three shapes: triangle, diamond, or square.

Input an odd number as the height of the desired shape.

Input a character to print for the shape.
