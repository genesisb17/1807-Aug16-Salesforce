There are comparisons to be made with the human body:

  • HTML is like the skeletal system. The bones, or rather elements, are structured and connected together.
  • JavaScript is like the nervous system. The brain is connected to nerves. It tells the body what to do and how to perform it. JavaScript tells the elements how to perform their tasks.
  • CSS is like the integumentary system. The integumentary system includes skin, hair, nails, and many other things. CSS is the design of the website. It has more compared to HTML on its own.
